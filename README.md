# TalentLoop

Plateforme de candidature intelligente — analyse CV, ciblage entreprises, lettres IA et simulateur d'entretien.

## Stack

- Next.js 15 (App Router)
- React 19
- TypeScript
- Tailwind CSS v4

## Démarrage local

```bash
npm install
npm run dev
```

Ouvrir [http://localhost:3000](http://localhost:3000)

## Pages principales

| Route | Description |
|-------|-------------|
| `/` | Landing page |
| `/candidate/recherche` | Campagne & candidatures IA |
| `/candidate/prix` | Tarifs |
| `/candidate/entretien` | Simulateur d'entretien audio |
| `/recruiter/dashboard` | Espace recruteur |
| `/onboarding` | Questionnaire profil |
| `/dashboard` | Tableau de bord candidat |

## Déploiement

Compatible Vercel — `npm run build`
